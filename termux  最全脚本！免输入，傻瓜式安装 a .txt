Termux 作为手机上最强终端，可以安装的软件很多，用 Python 爬虫，用 php 写博客，可以 nodejs 运行 vue……

功能强大不强大？强大！

玩法多不多？多！

不过，对于小白来说，这种“一切皆有可能”的强大能力反而成了一种桎梏，因为 Termux 太强了，不好掌控。

想用 php 搭建网站，只需要下载一个 ksweb，包含了 nginx+apache+php+mysql 一系列即开即用的工具。


想用 Python 写个小爬虫，只需要使用 Pydroid，内置包管理器，一键安装省心省力。


想用 Java 写个程序，只需要用 Jvdroid，自带 Maven 管理器，好用到爆。


可如果要用 Termux 来运行 php、python、java 等小程序，首先要搭建环境，配置文件，更多时候还要自己编译安装，最抓狂的是时不时就飘红报错……


很多小白满腔热血都倒在了环境搭建这一关，连门都没踏进去。


今天，我在这里给大家强烈推荐个易用性极强的 shell 脚本，可以帮助大家快速完成 Termux 的初始配置和日常管理，一键式搭建编程环境。


stone 管理工具
stone 是基于 Termux 的一款管理工具，操作简单，安装方便，集成了许多常用软件，还可以用 gui 的方式操作，非常适合新手来使用。


▌快速安装
你可以直接在Termux内输入以下命令安装本工具：


pkg i -y wget && bash -c "$(wget -qO- 'https://gitee.com/st1020/Stone-Termux-Tool/raw/master/st.sh')"


▌1. 切换清华源
Termux 自带的官方源不在国内，使用起来相当慢，动不动就出错， Stone 提供了一键切换国内清华源的选项，再也不用看着一堆代码发愁了。


▌2. 初始化
新手安装 Termux 后，看着黑乎乎的命令行一脸懵逼。

什么命令啊，脚本啊，记不住啊，根本不懂好不好！！！

Stone 给新手提供了初始化的脚本功能，免去了手动输入代码的麻烦。

刚刚安装的 Termux 不能访问手机内部存储，使用一键式 stone 脚本后会在 Home 目录下生成 storage/shared 软链接，目标指向了内部存储。

▌3. 安装常用工具
Termux 安装软件和安卓端不同，并不是下载了就完事了，安装后还需要配置文件，况且对于新手来说，光是找到这个软件就是个大问题。

stone 贴心的准备了 15 款常用软件，只需要输入对应的序号就能自动安装，纯傻瓜式的操作，易用性极强。

Nginx+Apache 这两款 web 数一数二的服务器软件，后台开发绝对不能少。

直接在手机搭建一个 hexo 博客网站。

Aria2下载工具吊打 ADM，下载速度快的一比。

kod 可道云可以在浏览器端管理 Termux 内部文件。

▌4. 搭建开发环境
linux 是编程最佳平台，稳定性极强。

Stone 同样提供了一键安装各类编程环境的脚本，支持 Python、C/C++、Node.js、Mysql、PHP、 Go、Java 等主流开发环境。


▌后记
Stone 的功能不止于此，需要大家一起去探索。

这里建议大家把它安装到 Termux 内部，之后只需要在命令行输入 st 就会自动启动该脚本，并打开 stone 管理工具。

对了，如果觉得命令行的方式不好看，还可以使用 GUI 模式来打开 Stone 管理工具，只需要输入命令：

st gui

就会跳出来一个充满现代化气息的面板。

是不是感觉瞬间回到了 Windows xp 时代。

而且，gui 模式可以用触控方式来操作。


除了 GUI，Stone 工具支持以下参数：

st [参数] 

参数	
无	直接进入管理工具
gui	进入本工具的GUI模式
install	安装本工具到Termux系统中
remove	删除本工具
update	更新本工具
help	显示帮助
当选择 install 命令时，可以指定一个参数，表示安装后运行本程序的命令，如不指定，默认为st。
